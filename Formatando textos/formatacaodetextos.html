Sobre as tags

Tag i deixa o texto em itálico.

Tag bold deixa o texto em negrito porém de forma não semântica.

Tag u deixa o texto com underline.

Tag mark deixa o texto com uma cor tipo marca texto.

Tag sup deixa algo acima da linha de texto como em 22.

Tag sub deixa o algo na parte de baixo da linha de texto como em H2O.

Tag blockquote transforma o texto em uma citação, dando um destaque através dos parágrafos.

Tag strong é a forma semântica de deixar um texto em negrito.

Tag font

A tag font usada dentro do html precisa de atributos.

Os mais comuns são:

  • color
  • face
    • É o tipo de fonte que será usada.
